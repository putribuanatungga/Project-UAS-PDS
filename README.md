# Project-UAS-PDS
Repository ini berisi analisis eksploratif dan penerapan metode supervised machine learning untuk klasifikasi dataset penumpang Titanic. Proyek bertujuan memprediksi status keselamatan penumpang berdasarkan karakteristik demografis dan informasi perjalanan base Python, meliputi praproses, eksplorasi data, perbandingan model, dan evaluasi kinerja.
